2.取得一份開放程式碼的 Unix 作業系統.學習它

我會假設你擁有（或能使用）一臺電腦。（花點時間欣賞的這意義有多大，駭客文化最初時， 電腦是如此昂貴，個人不能擁有它們的演變） 想要駭客的技能剛開始最重要的是得到 Linux  或 BSD-Unix 系統，並執行它。

除了 UNIX 外，當然還有很多其他作業系統， 但它們都是以二進位制形式分發的 — 你不但不能閱讀 那些程式碼， 你也不能改動它們。在 DOS，WINDOWS 或 MAC 等作業系統上從事駭客活動， 像穿著盔甲學跳舞一樣。
在  Mac OS X 是可能的，但只是系統的一部分是開源的 - 你可能碰到的困難，你必須要小心，不要取決於發展蘋果公司的專有代碼的壞習慣。如果你專注於 Unix 的引擎下，你可以學到一些有用的東西。
UNIX 是網際網路的操作系統。雖然你可以不知道 UNIX 使用網際網路的方法，你不能不理解 UNIX 。出於這個原因，今天的駭客文化由相當強烈的 Unix 為中心。（在以前並不是真的，一些舊時期的駭客仍然很不喜歡，但已經變得足夠強大，即使微軟似乎沒有嚴重削弱它的 UNIX 和網際網路之間的共生。）
所以請安裝一套UNIX — 我個人喜愛 LINUX 但其實有其他種類的(你可以同時安裝 LINUX 及 DOS /  WINDOWS 在 同一電腦上). 學習它.使用它. 把玩它. 用它來上網際網路. 閱讀及改動它的程式碼。它比微軟等的作業 系統提供更好的程式工具(包括 C .  LISP .  Python 及 Perl )。 比任何微軟操作系統主機好用，你就會有樂趣，你會吸收更多的知識並你意識到你正在學習，直到你回頭看看它已成為一個黑客高手。
要知多點關於學習 UNIX 的資料請到 Loginataka。你可能也想看看 UNIX 編程的藝術。
為了讓您的雙手在Linux，看 Linux 網站 ;你可以從那裡下載或（更好的主意）找到一個本地的 Linux 團隊，以幫助您進行安裝。
在首十多年的這份 " HOWTO " 的生活，我報的是從一個新的用戶的角度來看，幾乎所有的 Linux 發行版等同。但在2006-2007年，實際的最佳的選擇出現了：Ubuntu Linux系統。其他發行版都有自己的優勢領域，Ubuntu 是距離遠，最容易給 Linux 新手。要小心，雖然的猙獰與夜間無法使用的“統一”的桌面界面，Ubuntu 的介紹，作為一個默認的幾年後的的 Xubuntu 上或 Kubuntu 的變種是更好。
你可以找到 BSD Unix 的幫助和資源在 www.bsd.org。
你的腳趾浸在水中的一個好辦法是啟動了 Linux 愛好者稱之為現場 CD，分佈完全從 CD 運行，而無需修改您的硬碟 disk.This 將是緩慢的，因為 CD 是緩慢的，但它是一種方式來獲得，而無需做任何激烈的可能性看看。
我已經寫了基礎上 UNIX 和網際網路。
我建議不要安裝 Linux 或 BSD 作為一個單獨的項目，如果你是一個新手。現在安裝已經變得夠好了，這樣做完全是可能在自己的，即使是新手。儘管如此，我還是建議您聯繫您當地的Linux使用者群尋求幫助。它不能傷害，和可能順利的過程。

3.學習如何用網際網路及用 HTML 來寫網頁
大部份駭客所做的事情, 如在工廠及辦公室或大學裡的工作, 是不會對非駭客造成影響的. 但網際網路卻是一個例外, 就算政治家都承認互聯網是一個足以影響世界的東西.亦由於這原因(及其他種種原因)你要學如何在網際網路上工作.
　 這不是說你要學如何使用瀏覽器(人人都懂使用..), 我是說你首先要學用HTML來寫網頁. 若你根本不懂寫程式,用HTML來寫網頁可幫助你得到一些程式編寫的基本概念.
　 可是寫網頁仍不足夠, 在網際網路上有很多網頁, 但大部份是沒有方向, 明顯地沒有內涵的網頁. (你大可到 The HTNL hell page 看看)
　 所以, 你的網頁一定要有內容—必須是有趣的及/或有用的(對其他駭客), 這便是我在下一個題目要說的..



4.如果你不具備好的英語，學習它


由於身為美國人和英語為我的母語，我以前一直不願表明，免得它被視為一種文化帝國主義。但一些其他語言曾促請我指出，英語是駭客文化和網際網路的工作語言，你需要知道它的用在駭客社群。

早在 1991 年左右，我了解到，許多駭客使用英語作為技術討論，即便當他們有同樣的非英文母語。我看到一個報告說，比起其他語言英語具有更豐富的技術詞彙，所以它就是一個更好的工具來完成工作。出於類似的原因，用英文寫的技術書籍翻譯往往不如人意（就算他們被徹底翻譯）。

Linus Torvalds，芬蘭人，他用的英文為他的程式碼寫注解（他顯然的沒有用英語除外的語言寫他的注解）。招募世界各地的開發者社群，來開發 Linux，他流利的英語一直是重要因素。這是一個值得學習的榜樣。

作為一個以英語為母語的人並不能保證你有良好的語言表達能力，足以充當駭客。如果你的寫作是半生不熟.不合語法和錯字連篇，許多駭客們（包括我自己），往往忽略了你。雖然馬虎寫作並不總是意味著馬虎的想法，我們通常會發現，他們之間的相關性強 - 草率的思想對我們沒有幫助。如果你還不能勝任英文書寫，就快點學習。


駭客社群的身份
1. 編寫公開程式碼的軟件
2. 幫助他人測試及除錯公開程式碼的軟件
3. 發行有用的資訊
4. 幫助系統運作
5. 為駭客社群服務
像大部份非謀利的社群， 駭客以名譽為重. 你會嘗試解決有趣的問題, 但問題是否有趣及你的解決方法是否有效， 卻只有其他具有技能的駭客們可以去品評.
因此, 要玩駭客這個遊戲, 你要以其他駭客對你的觀感作為自己的評價。(正如我說你不是駭客,除非其他駭客亦稱你為駭客)。不過這事實卻有時被人誤解, 因為他們多以為駭客是孤獨地工作的人。

人類學家稱駭客為貢獻社群. 駭客得到地位及名譽並非來自強於他人的權力，亦非由於外表美麗，或有異於常人的能力，而是源於施與。清晰地說，是將你的時間，創意及創意的結果貢獻他人。

基本上， 你可以貢獻 5 方面的東西以獲得其他駭客的尊重：

1. 編寫公開程式碼的軟件
第一個（最關鍵和最傳統的）是寫其他駭客認為有趣或有用的程序，給程序源遠到整個黑客文化使用。
（我們習慣於稱這些作品“自由軟件”，但這種困惑的人太多了，誰不知道究竟應該“免費”的意思，我們大多數人現在更喜歡使用“開放源碼”軟件）。
駭客道最受尊敬的半神人誰寫大，有能力的方案，遇到了一個廣泛的需要，並給予他們走，所以現在每個人都使用他們。
但這裡有一個優良的歷史點位。雖然黑客們總是抬頭其中開放源碼開發作為我們社會的最堅硬的核心，在20世紀90年代中期之前，大多數的駭客們大部分時間從事閉源。這仍然是真實的，當我寫這個HOWTO的第一個版本於1996年花了1997年後的主流開源軟件改變的事情。今天，“黑客社區”和“開源軟件開發”兩個什麼本質上是相同的文化和人口的描述 - 但它是值得記住的是，這並非總是如此。 （在此的更多信息，請參閱本名為“歷史注：黑客，開源和自由軟件”）。
2. 幫助他人測試及除錯公開程式碼的軟件
他們還充當誰的立場和調試開源軟件。在這個不完美的世界中，我們難免會花費我們的軟件開發時間在調試階段。這就是為什麼任何開放源碼的作者誰的思想會告訴你很好的測試，測試者（誰知道如何描述清楚症狀，本地化的問題很好，可以容忍匆匆發布中的錯誤，並願意套用一個幾個簡單的診斷例程）都值得他們在紅寶石的重量。即使這些都可以使調試階段之間的區別，這是一個長期的，累人的噩夢，這僅僅是一個有益的滋擾。
若你是初學者， 你可以找一些正在開發階段而你又感興趣的程式來做測試及除錯。 當你懂得測試及除錯，你自然會學習更改程式。這樣你不單可以學習， 也可以與其他專家建立良好的關係，而他們亦很可能會於未來給予你幫助。

3. 發行有用的資訊
另外一點是你可以收集及挑選有用的資訊並製成網頁或 FAQ (常問問題)， 以供諸同好技術性 FAQ 的經營者往往很多人的專重， 如同編寫公開程式碼的軟件的專家一樣。


4. 幫助系統運作
駭客社群(特別是互聯網的發展者)是由自願者組成的。 當中有大量必須但平淡的工作須要有人去處理 -- 例如管理電子郵件，監察討論小組，經營大型程式的儲存網站，及發展技術性標準等。
　 
做以上工作的人得到很多人的專重， 因為人人都知道此種工作是很花費時間但又不如編寫程式般有趣。此等工作須要使命感。

5. 為駭客社群服務
最後，你也可以成為傳播文化本身（例如，如何成為一名駭客，寫一個準確的引物:-)。）這是不是你會被定位於做，直到你已經同時左右，成為眾所周知的第一個四件事情之一。
事實駭客文化並沒有領袖，但它確實有培養英雄和部落長老和歷史學家和發言人。當你一直在戰壕中足夠長的時間，你可能會成長為其中之一。當心：駭客們的不信任在他們的部落長老赤裸裸的自我，所以明顯地達到這種成名是危險的。 ，而不是追求它，你有自己的位置排序，所以滴在你的腿上，然後你的情況是溫和和親切的。
駭客與怪客的關係
與流行的傳說相反，你不必是個怪客而是一個駭客。它有幫助，但是，很多駭客們實際上是怪客。作為一個社會拋棄的東西，可以幫助你保持集中在真正重要的事情，喜歡思考和盜號。

出於這個原因，許多駭客已經採用了標籤'怪胎'驕傲的徽章 - 這是一個方式從正常的社會期望宣布其獨立性（以及其他的東西像科學小說和戰略遊戲，經常去的是一個愛好駭客）。術語“書呆子”要使用這種方式早在20世紀90年代，回來的時候是一個溫和的“怪客”輕蔑和相當嚴厲的'怪胎'，2000年後的某個時候，他們交換的地方，至少在美國流行文化中，有現在甚至是顯著的怪胎人誰不是技術人員之間的文化自豪感。

如果你能集中足夠盜號要善於仍然有自己的生活，這很好。這是一個容易得多相比，今天當我還是個新手，在20世紀70年代，主流文化是更友好的技術怪客現在。甚至有越來越多的人意識到，駭客們往往是高品質的戀人和配偶材料。

如果吸引你的駭客，因為你不能有自己的生活，這是確定了 - 至少你不會有麻煩集中。也許你會得到一個生命以後。

Ｑ 我的系統剛被破解. 你會否幫我防範再被破解 ?

Ａ 不會。 每次我收到這類要求的，都是用視窗作業系統的。基本上是不可能有效地防止視窗作業系統被破解;因為這作業系統有太多的程式及結構缺點，就像要用篩子來將船划動一樣。只有轉用 Linux 等有真正安全的作業系統方可解決這個問題。

Ａ 是的。進入到 DOS 提示符，然後鍵入“格式 c：”。您所遇到的任何問題，將在幾分鐘內不見。

Ａ 最好是找一個你本區的Unix或Linux的討論小組, 並參與討論(你可以在 LDP 的網頁找到超連結)我曾說在IRC(網上即時討論小組)不能找到真駭客, 但我發覺現在情況有所改變. (現時有些真駭客亦在GIMP及Perl等的網上即時討論小組出現.)
